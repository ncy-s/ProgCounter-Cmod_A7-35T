# ProgCounter-Cmod_A7-35T
A simple 4-Bit Verilog Program Counter for the Xilinx CMod A7-35T, iterates from 0000 to 1111 before overflow and the pattern repeats.

**Hardware Setup:**


**Inputs**

CLK (Clock): Pin 3 (A16)

CE (Chip Enable): Pin 5 (C15)

RST (Reset): Pin 7 (A15)

**Outputs**

Bit 3: Pin 23 (P1)

Bit 2: Pin 22 (N2)

Bit 1: Pin 21 (N1)

Bit 0: Pin 20 (M2)
